# A + B Problem

A template problem for VerdeJudge.

## File Structure

- `problem/{id}` folder
    - `README.md` the description file
    - `problem.yml` problem metadata
    - `testcases`
        - `hello_{n}.in`
        - `hello_{n}.ans` testcases
    - `scripts` (todo)
        - `solution.cpp` the solution code
        - `checker.cpp`
        - `interactor.cpp`
        - `validator.cpp`
        - `generator.cpp`

## Description

Given two integer $A$ and $B$, output their sum $A + B$
